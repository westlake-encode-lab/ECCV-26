ECCV 2026 Tutorial

Efficient MLLM Inference via Approximated and Exact Computing.

The tutorial is separate from the paper track. It provides a structured overview of efficient MLLM inference, from approximate methods that reduce redundancy to exact system methods that preserve model outputs.

Date Sep 8, 2026
Time Morning Session
Location Malmö, Sweden
Tutorial Page
Approximate Computing Reduce redundant computation

Methods that lower inference cost through controlled approximation.

Exact Computing Preserve model outputs

System techniques that improve serving efficiency without changing predictions.

Program Structure From compression to serving
Part I

Part II

Token Efficiency

Compression methods for long multimodal contexts and autoregressive generation.

Part III

System-Level Design

Serving and hardware-aware optimization for throughput and latency.
